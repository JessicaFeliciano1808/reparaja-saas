# Roadmap do Produto – ReparaJá SaaS

## Fase 1 – MVP SaaS
Objetivo: transformar o ERP atual em SaaS funcional.

Funcionalidades:

- Multi-tenancy (multi-oficina)
- Cadastro de oficinas
- Login de oficinas
- Isolamento de dados por oficina
- Portal do cliente
- Aprovação de orçamento online
- PDF de orçamento
- Dashboard básico

Status: Em desenvolvimento

---

## Fase 2 – Automação de Oficinas

Funcionalidades:

- Integração WhatsApp automática
- Notificação de status do veículo
- Histórico completo por veículo
- Controle de peças usadas
- Controle de estoque avançado
- App mobile para mecânicos

Objetivo:

Automatizar comunicação com clientes e controle de serviços.

---

## Fase 3 – Plataforma SaaS Completa

Funcionalidades:

- Dashboard SaaS global
- Relatórios financeiros avançados
- Controle de planos e assinaturas
- Sistema de cobrança automática
- API pública
- Integrações externas

Objetivo:

Transformar ReparaJá em plataforma SaaS escalável.

---

## Fase 4 – Escala Nacional

Funcionalidades:

- Marketplace de peças
- Integração seguradoras
- Integração locadoras
- Integração despachantes
- Inteligência artificial para diagnóstico

Objetivo:

Tornar o ReparaJá o principal sistema de gestão para oficinas do Brasil.

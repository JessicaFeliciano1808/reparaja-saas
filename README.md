# ReparaJá SaaS
Sistema ERP para oficinas automotivas.

## Funcionalidades
- Gestão de clientes
- Gestão de veículos
- Ordens de serviço
- Portal do cliente
- Controle de estoque
- Financeiro
- Relatórios
- Integração com WhatsApp

## Tecnologias
- React + Vite
- Node.js + Express
- PostgreSQL
- Drizzle ORM
- TailwindCSS

## Objetivo
Transformar oficinas automotivas em operações digitais com controle total de serviços, clientes e faturamento.

## Roadmap
- Portal de aprovação de orçamento
- Integração com WhatsApp
- Sistema SaaS para oficinas
- Cobrança de mensalidade automática

# 🚀 ReparaJá SaaS
Sistema ERP completo para oficinas automotivas — transformando operações manuais em digitais.

---

## 🔹 Funcionalidades
- 👥 Gestão de clientes
- 🚗 Gestão de veículos
- 🛠️ Ordens de serviço
- 🌐 Portal do cliente
- 📦 Controle de estoque
- 💰 Financeiro
- 📊 Relatórios avançados
- 📱 Integração com WhatsApp

---

## 💻 Tecnologias
- ⚡ Frontend: React + Vite + TailwindCSS
- 🖥️ Backend: Node.js + Express
- 🗄️ Banco: PostgreSQL + Drizzle ORM
- ☁️ Deploy: Frontend Vercel, Backend Render

---

## 🎯 Objetivo
Transformar oficinas automotivas em operações digitais com controle total de serviços, clientes e faturamento.

---

## 🗺️ Roadmap
- 📄 Portal de aprovação de orçamento
- 📱 Integração com WhatsApp
- 🏢 Sistema SaaS multi-oficinas
- 💳 Cobrança de mensalidade automática
